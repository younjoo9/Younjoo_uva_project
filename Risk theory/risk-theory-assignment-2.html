2 Makeham’s mortality law

# Survival function for the Makeham distribution (a≥0, b≥0, c>1)

sMakeham <- function (x, a, b, c) {
  ifelse(x<0, 1, exp(-a*x-b/log(c)*(c^x-1)))
}

Question 3(a)

dMakeham <- function (k, a, b, c) {
  ifelse(k>floor(k), 0, (sMakeham(k,a,b,c)-sMakeham(k+1,a,b,c)))
}

Question 5(a)

If we use ‘allages’, our computation implies that all people has died on their birthday. This is unrealistic.

We should consider that people could pass away at any moment, and adjust our computation.

One option we can choose is to assume Uniform Distribution of Deaths(UDD) in each year of age. Under UDD, we can say that people who died between the age \(x\) and \(x+1\) have averagely survived for \(x+0.5\) years.

Therefore, using ‘allages+0.5’ is reasonable rather than ‘allages’.

6 Reducing Makeham’s law to Gompertz’

Question 9(a)

# Three functions for Gompertz distribution

sGompertz <- function (x, b, c) {
  ifelse(x<0, 1, exp(-b/log(c)*(c^x-1)))
}

dGompertz <- function (k, b, c) {
  ifelse(k>floor(k), 0, (sGompertz(k,b,c)-sGompertz(k+1,b,c)))
}

negloglikGompertz <- function (bc) {
  -sum(d.x[ages+1] * log(dGompertz(ages, bc[1], bc[2])))
}

# Minimizing negloglikGompertz

Thus, we assume that Makeham fits better.
